SwissArmyPCD
Este repositório contém uma aplicação moderna desenvolvida com React, Vite
e TypeScript, utilizando Tailwind CSS e shadcn/ui para oferecer uma
experiência visual acessível, responsiva e de alta performance.
O projeto pode ser editado facilmente em qualquer ambiente local de
desenvolvimento.
Tecnologias Utilizadas
 React
 TypeScript
 Vite
 Tailwind CSS
 shadcn/ui
Essas tecnologias foram escolhidas para garantir rapidez no desenvolvimento,
fácil manutenção do código e excelente desempenho em produção.
Instruções de uso:
git clone https://github.com/eliphaslevii/TrabalhoFinal-IHC
cd TrabalhoFinal-IHC
npm install
npm run dev
O servidor de desenvolvimento inicia com recarregamento automático,
permitindo visualizar mudanças em tempo real.
Aguarde o build e obtenha o link de produção
